# Pre-requisites

* PostgreSQL: `brew install postgresql`
* Heroku Toolbelt: `https://toolbelt.heroku.com/`
	- Or, at least, foreman: `https://github.com/ddollar/foreman`

# Installing

```bash
make
```

# Testing

```bash
rake spec
```

# Continuous testing

```bash
guard
```

# Running

```bash
make run
```
